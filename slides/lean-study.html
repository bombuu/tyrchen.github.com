Learn a markup language

I use markdown for blogging, reST for my book

Why markup language?

  • richer than pure text
  • write once build many
  • could be managed via SCM - e.g. git
  • Google it yourselves to find the grammar

Systematically build knowledge with Sphinx

Why Sphinx?

  • write in reST, build to 22 formats (including html and pdf)
  • build cross reference, ToC, etc. automatically
  • let you focus on content itself

How to use it?

  • install python, then install Sphinx
  • to generate PDF, install latex
  • sphinx-quickstart will help you with bootstrap
